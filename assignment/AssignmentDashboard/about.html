About Dashboard:

This dashboard is used to track about the each project status.

There are 4 columns: Backlog,Ready,In-progress,In-review

Backlog: It is a work that needs to be completed.

Ready: It is in the usable state.

In-progress: It means it has started and is still continuing.

In-review: It means it is in the evaluation stage

How it Works:

TO CREATE: click on the +Add item buttons to create new card where u can write.

TO SAVE : click on the SAVE button to save the content u have written into firebase.

TO EDIT : click on the UPDTAE button to update/edit your text and save .

TO DELETE : click on the DELETE button to delete the saved card .
