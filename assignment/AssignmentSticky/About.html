About Sticky Notes:

Sticky notes are used as reminders. Sticky notes can be used for project-related work or non-project related work.

A great way for you to make some brief notes on a topic you are studying or working on

For example, a sticky note can be created to a human task reminding the developer to email a document to a manager or for note down the important stuffs to do.

How it Works:

HOW STICKIES ARE STORED HERE : stickies are saved to browser local storage. You can refreshthe browser stickies will be rememebered. Deleting browser history however, will remove them.

TO CREATE STICKY : click on the + new note buttons to create new sticky.

TO SAVE STICKY : click on the SAVE button to save the text of the sticky.

TO EDIT STICKY : click on the EDIT button to update your text and save the text of the sticky.

TO DELETE STICKY : click on the DELETE button to delete the sticky note .
